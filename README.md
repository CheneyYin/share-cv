## 安装

```shell
pip install -r ./requirements.txt
```

> 建议使用虚拟环境，例如`python -m venv ./env`来创建虚拟环境，然后**激活环境**，可参考这里([venv --- 虚拟环境的创建 — Python 3.12.6 文档](https://docs.python.org/zh-cn/3/library/venv.html))。

## 执行演示代码

请在项目根目录下，执行如下命令。

```shell
python .\src\traffic_analysis.py
```
